This is a Flight Price Prediction System . Which is base full EDA ,Model Training  and connected Dataset using pie chart. 
 # Flight Analysis Dashboard ✈️
An interactive Power BI dashboard designed to analyze flight booking data, pricing trends, and performance across different airlines and months.

## 💡 Key Insights Uncovered
* **Total Volume:** Analyzed a total price distribution of **97M** across different flights.
* **Top Performing Airline:** **Jet Airways** dominates the market share with a total price contribution of **45M**.
* **Monthly Pricing Trends:** 
  * Experienced a significant drop in bookings/price during **Month 4 (6M)**.
  * Reached a peak performance in **Month 5 (32M)**, followed closely by **Month 6 (30M)**.

## 🛠️ Tech Stack & Skills Used
* **Tool:** Power BI Desktop
* **Data Transformation:** Power Query
* **Data Modeling:** Star Schema / Relationships
* **Visualizations Used:** Bar Charts, Pie Charts, Slicers, Card Visuals

## 📂 Project Structure
* `flight_dataset`: Raw data used for analysis.
* `Flight Analysis Dashboard.pbix`: Main Power BI project file.
